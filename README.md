# Metal Wealth Tracker

A minimal, server-side rendered application to track the value of Gold and Silver holdings. Built with **Astro** and hosted on **Cloudflare Pages**.

### ⚠️ Disclaimer & Liability
**This project is for educational and demonstration purposes only.**

1.  **No Financial Advice:** The data and calculations provided by this application are estimates and should not be used for trading or financial decisions.
2.  **Data Accuracy:** Market data is fetched from free, public APIs and may be delayed, inaccurate, or unavailable.
3.  **No Data Storage:** This application does not use a database. No user data, input values, or holdings are stored or transmitted to the server. All inputs remain in the user's browser session.
4.  **No Warranty:** The software is provided "as is", without warranty of any kind.

### Tech Stack
* **Framework:** Astro (Server Side Rendering)
* **Hosting:** Cloudflare Pages
* **Data:** cdn.jsdelivr (Currency) & Yahoo Finance (Sparklines)
* **Styling:** Native CSS (No framework)
